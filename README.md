### Generazione Utenti

La generazione degli utenti avviene grazie a una funzione chiamata LoadData. Quest'ultima è una funzione  async, così da permettere il funzionamento dell'await. Infatti alla funzione SendRequest è posto davanti il 	costrutto await che aspetta che la funzione a cui è assegnato sia finita così da permettere che il server di 	randomu user risponda e così di restituire i dati.
